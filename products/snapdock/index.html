SnapDock — Markdown Workspace

A clean, fast Markdown reader built with pure JavaScript, HTML, and CSS. SnapDock provides a distraction-free workspace with file-tree navigation, multi-file viewing, and a polished ZFordDev interface.

View Source Download Latest

Key Features

File-Tree Navigation

Browse folders of Markdown files with a simple, responsive sidebar.

Multi-File Viewing

Open multiple documents and switch between them instantly.

Drag-and-Drop Support

Drop files or folders directly into SnapDock to load them immediately.

PDF Export (Planned)

Future support for exporting Markdown documents to clean, styled PDFs.

Engineering Notes

SnapDock is intentionally lightweight and dependency-free, built with native web technologies for maximum clarity and maintainability. The UI follows ZFordDev design principles: clean, minimal, and distraction-free. Cross-platform builds are planned once the underlying OS tooling is ready.
