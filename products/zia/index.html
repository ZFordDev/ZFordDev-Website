ZIA — Modular AI Assistant Framework

A proof-of-concept modular AI assistant that connects Slack, Discord, and a web interface to local or external AI models. Built to explore persona switching, persistent memory, and multi‑platform routing. ZIA now serves as the foundation for ALX‑ZIA — the next-generation hardware-first evolution of this architecture.

View Source Learn About ALX

Key Features

Slack & Discord Handlers

Unified message routing across multiple chat platforms.

Persona Templates

Switch between conversation styles using JSON-based persona configs.

Persistent Memory

Per-channel JSON memory storage with no database required.

Config-Driven Setup

All behavior controlled through JSON files — no code edits needed.

ALX‑ZIA Integration

ZIA is the software backbone that inspired and now connects to ALX‑ZIA, the hardware-first successor.

Engineering Notes

ZIA is intentionally raw — a prototype that laid the groundwork for ALX‑ZIA and ZiaStudio. It demonstrates routing, persona switching, and memory, but is not intended as a polished product.
